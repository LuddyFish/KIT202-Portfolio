Assignment 1

I am quite happy with how my group's assignment 1 turned out. It is a beautiful, yet simple and clean blogging site that has it's own personality.
Working together was a little challenging in the sense that all our group members understood what needed to be done, but it also removed a lot of communication as we worked a lot off of vibes and a lot of trust, which did turn out well, and we rarely needed each other's help when we got to work on certain tasks. We did often say what we had completed to each other in our Discord group chat and we did talk about ideas during our tutorial class together. The only real downside to us not working more collaboratively was that we didn't really know how each other's page designs functioned, so unless we took the time to look at them for ourselves, we were going on full trust mode. We did manage to create our baseline standards from the beginning, such as colour choices, fonts, and formatting, but some inconsistencies did slip through like certain buttons not being the same colour. But other than those few things, I would believe that we were generally ok and fine with the level of communication we had.
What I found most interesting with using HTML, CSS, and JavaScript to develop the blog was how much freedom and possibilities we had. There's still a few more design choices that I'd like to implement for assignment 2 just because we didn't have the time to implement everything. From what I've experienced, HTML is the skeleton to it all. Add in CSS, and you can shape and modify the contents to your heart's content. I didn't even realise animation was possible with CSS, which was fun to play around with. And to top it off, JavaScript being able to read and change the design of things based on user inputs makes it a whole lot more interactive. With the three of them, any website design is virtually limitless, and it impresses me that you can design any website for visual purposes with just them.
The only thing left is to do the backend where all the data is stored.
