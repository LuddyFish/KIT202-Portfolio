This topic introduced me to the concept and use of HTML and CSS (and JavaScript to an extent). I felt overwhelmed initially when I started working with HTML due to the amount of new things to learn, but things started to become a lot more clear once I got the ball rolling and realised that it wasn't as initimidating as it appeared to be. The first week keeping everything to just HTML made it real easy to get into.
The fun part though was learning how to use CSS as it allowed me transform sites to any style and form I wanted. There was a lot to cover and make sense, but seeing the near limitless potential of CSS, I realised that you could make a very decorative and still very clean site.

DB Docs

> Link

This topic introduced me to Entity-Relation (ER) Diagrams, which I had some past experiences with in KIT102 (when it still existed). The concepts were similar and much more broken down, so nothing was too challenging. It was mostly a refresher into thinking and planning ahead for databases.
This was also my first glimpse into phpMyAdmin, where we could implement the base foundations for ER Diagrams.

PHP Intro

> Link

This topic introduced the language of PHP, which in the large scheme of things functions similarly to JavaScript. The main difference is that PHP has a lot more freedom but at the cost of clarity as it is very easy to lose track of where things are. This has taught me that while PHP can be used in nearly every JavaScript case, you're better off using JavaScript in the places it can work to avoid reability and accessibility problems.
